Alphacash
=====================

Intro
-----
Alpha is the native currency for Unicity a platform for building decentralized
application using Verifiable Autonomous Agents. 


Setup
-----
Unpack the files into a directory and run alpha-qt.exe.

Alpha is the original Alpha client and it builds the backbone of the network.
However, it downloads and stores the entire history of Alpha transactions;
depending on the speed of your computer and network connection, the synchronization
process can take time.

See the Alpha docs at:
  https://github.com/unicitynetwork/alpha
for more help and information.
